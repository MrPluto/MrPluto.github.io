下一站，杭州

去年某个周末和深圳的同学们约在我家, 酒过三巡, 谈天说地, 脑海中闪过念头, 去杭州看看吧。于是年初, 我便来到了杭州。

来杭州前, 我度过了几天自在的日子。 原计划2月底离职, 然后给自己放几天假, 3月中旬才启程杭州, 然而公司了无生机的工作气氛让我再也无法忍受, 第三天我便提出的辞职申请, 交接完工作。离职后的几天, 关掉闹钟, 早上柔柔的阳光洒进床头, 打开音乐, 在租来的20平小房子悠哉的晃悠的, 有时下午便骑着小黄车, 在周围转着, 时时吹来的微风带着海咸咸的味道; 又或者把椅子搬到阳台, 静静的看看书, 看看湛蓝的天空。


其实离开深圳是有些不舍得的。 深圳的冬天很舒服, 家里人人都里外三层的围炉拥衾, 而我穿个长袖就能到处晃荡; 周围的同学也挺多, 虽然我更喜欢一个人, 但能有一些人来去都方便, 对于刚入社会的人是有当时无法察觉的帮助的; 据住处只有两个地铁站的图书馆也是我很难割舍的, 在很多个难熬的夜晚, 都是它给予着我无尽的力量; 一线的互联网文化让我们可以最先享受它带来的方便和新鲜, 潮汕美食也让人垂涎。

然而我感觉我不属于这里, 也许是我没花时间来认识这座城市, 也许是它缺少一些历史让我慕名, 也许它是座没有四季的城市, 没有四季, 也就没有交替, 没有变化。也许是我自己也不知道我想要的是什么, 也许我寻找的在其他地方。

比如杭州

年前开始打开网上的简历状态, 其实并不急着, 也没有太多的关注着, 一家公司找到我,也是约在年后再交流, 未曾想刚回到深圳就来了电话,问我的计划, 后来又给我电话, 前前后后5、6个, 想想其实反正要去杭州了, 于是便整理了家里的东西, 买好车票, 历时11小时, 到达了杭州。大学时来过杭州一次, 天气干冷, 其实老家也是这样的,只是在深圳习惯了温暖的天气, 一下子反倒觉得杭州比之前印象中更冷了。安顿下来后,剩下的就慢慢开始吧。

朋友们都焦虑着, 读研的不知自己想干什么, 老家稳定工作的不甘寂寞,出来后一时也不知所措,似乎很多人都不知道自己想要的是什么,都焦急,都忧虑,一个人时也忽然感叹着。理想状态,他时隐时现,他忽近忽远,在雾中,而我们在码头上,等待摆渡人。

发表于 2017-02-09

这是毕业后过的第二个春节

1月23号, 我坐上了回往家乡的火车。 深圳那时的天气还是很暖和的, 我只穿了件卫衣却仍有些燥热。 火车晚上6点准时到达目的地,迎接我的是家乡的风。得知表弟在一家艺术培训学校学习隶书书法, 转车接我表弟, 第二天同表姐一起回到家乡小镇。

老爸在沙发上看着书, 老妈去沿江风光带上散步去了, 我到家已经又是傍晚。

四方的朋友们也都回来了, 攒局吃饭喝酒打牌, 很久未曾见面, 甚至于电话也不曾有过, 然而都默契着仿佛往日时光。

见老朋友总是件高兴的事, 旧日的欢乐牵连的彼此, 饭桌上调侃着女朋友, 吹嘘着酒量, 插科打诨;牌局上互相挑衅, 出臭牌时的挖苦嘲笑, 个个都如妙笔生花,个个都意气风发着。局散更有明天, 春节假期是很长的。

除夕是在奶奶家过的, 之前的约定是每家一年轮着来, 但是奶奶身体没有那么硬朗了, 今年起都在奶奶家, 奶奶也不用舟车劳顿。

写到奶奶, 眼眶有点湿润了, 读书时有时候想起奶奶, 想到晚上奶奶一个人坐在电视机前, 灯好像也不是很亮了, 炉火似乎也不太热, 不然为何奶奶总是瑟缩着身子, 我总跟我自己说, 放假回去要多陪陪奶奶。除夕晚上, 一家人坐在客厅里, 奶奶坐在电视机旁, 就在我脑中想象的位置, 只是身子向着我们, 看着一家子人说笑, 说上几句话, 好像得不到大家的重视, 被淹没了。奶奶开始烧火了, 三十夜里的火, 烧到天明。奶奶包起了饺子, 我一向是不包的, 偶尔尝试一下, 包个不伦不类的出来算是了事。晚上大家聚着, 奶奶端出和好的馅儿,擀好的皮儿,坐到一角,默然的包着, 我看着很不舒服, 平时放假回家到奶奶家我是兴高采烈的, 奶奶脸上也挂着淡淡的笑, 但坐着的时候我感觉总是不能给奶奶更多,我感觉我输出不了,只能是挑问的村里的人和事。奶奶慢慢的包着饺子,腰上的旧疾使她站成奇怪的角度, 我一步挪过去,要跟奶奶学包饺子, 我也呼唤着表弟过来, 我不知道说些什么, 但是此时奶奶大概不孤独了吧。也许爷爷的灵魂也在旁边看着我们包呢。

除夕后的走亲戚, 不提也罢。只是表哥的准媳妇没见到有些可惜。

年后牌局攒得艰难了, 各处的婚礼, 我们也真的到了这个年纪了。从老妈三句话不离同事儿女结婚, 每日一个婆媳关系调研开始, 旁敲侧击着我了。

前天回到了深圳,回到了自己的窝。

我回想着, 在这个大节日里, 在这个匆忙又欢乐的假期里,是否做了我希望做的事,是否跟我希望的感觉一样,为什么我会这样的希望着?

除夕的烟花爆竹太红火热烈, 恍惚我了。

我的2016

发表于 2017-01-03

2016, 远方仍在寻找。。。

22号回家过年, 也表明了我在深圳的2016年彻彻底底的翻篇了。写几句话聊以记忆。

当然还记得年初2月份来深圳时, 和 Victory 在车公庙的一家咖啡店, 我们喝着浓浓的咖啡, 互相说着新年回家时发生的趣事, 后来聊到这新的一年–2016年, 我心里满是期待和不安分, 觉得这一年将会很精彩, 会收获很多, 学习很多。 那时我已经准备辞职离开我的第一家公司, 重新开始找工作。 他也给了我一些工作方向上的意见。总之我觉得很好。

关于工作, 其实并没有什么值得记录的, 自己一个人慢慢的摸索的新的技术, 很慢, 但是也乐在其中, 在互联网计算机世界里, 确实能让人时时发现有趣的事, 好玩的事。

关于读书, 我其实算是半个爱书之人, 喜欢读书, 喜欢沉浸读书时那种忘我的思想环境。但是这对我也算种奢求, 因为我自己总是心神难宁, 心里总是有东西在挠, 这一年读书甚少, 看看图书借阅记录和微信读书记录, 一共读了不过40本, 其中也有一些是未读完的。不过值得高兴的是, 我认识了刘震云、萧红、约翰·威廉斯、东野圭吾, 这让我觉得很满足。

关于音乐, 年初在朋友推荐下开始用网易云音乐, 从此发现了一众歌手, 我是喜欢民谣的, 无论歌词还是旋律, 都让我觉得畅快。 在网易统计下累计听歌4000多首,也算不枉一个爱音乐的人。

关于生活, 我是一个人住的, 我喜欢独处, 但却常常不知道如何来独处, 如何来面对自己的内心和欲望。无论读书,音乐,电影、跑步, 我是一直在寻找属于自己的生活方式, 一直在尝试着, 有时缺少了些勇气, 在舒适圈里不肯出来, 内心在拒绝着外界的好与不好, 有些顽固的在自己格物, 以求致知。我是该牺牲掉一些安全感, 来开始真正的拥抱这个世界, 开始热爱她了。

到现在回想这一年, 跟我之前所希望着的确是有差距的, 无论工作还是生活。但我仍旧像之前的那样, 希望着, 郑女侠说她已经不需要远方了, 但是我还是想寻找下远方的。

看到火星情报局一期里用一个字形容2016年, 我的就是 ‘寻’ 吧。

2017 我希望

  • 开始维护我的个人博客了,记录些自己所感所想,记录些有趣的书、电影和音乐。
  • 认真的体会一座城市
  • 带奶奶游一圈杭州,完成她的这个小小心愿
