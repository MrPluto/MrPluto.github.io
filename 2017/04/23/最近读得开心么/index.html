最近读得开心么

发表于 2017-04-23


今天是世界读书日，来说说我的一些小故事

我读小学时，跟我奶奶一起住在医院一隅的小食堂里，那时我奶奶已经退休了，负责给医院的员工提供早餐中餐。医院的同事有一个男孩 C 和我是同班同学，我们天天玩在一起，那时候很喜欢冒险，而恰巧有个儿童探险系列 《冒险小虎队》 很流行，哪一个同学如果买了，那他下课肯定会被团团围住，我们都抢着要借。可是那时零花钱可少得很，书又贵，不是每出一本新的就能买的。我和 C 于是每到中午就跑到医院旁边的书店里去看书，我们两个一屁股坐在地上，唰唰的看，中午时间很短，刚刚好能看到最后那几节就得撒腿跑回学校，这个系列的书每一小节最后都会有一道谜题要我们去解答，只有买书才能拿到那张解密卡，所以遇到解不出来的谜题时，就很心急了。这个系列一开始一共有 40 本，后来出了升级版的，书也厚了，解密卡也高级了，当然价格也贵了。

每到寒暑假都会跟妈妈一起到外婆家住上几天，外婆家房子很大，房间很多，最喜欢翻东西，某天在大姐姐书柜下一个侧箱里发现 3 本黑白相间封皮的书， 封面上一个人像，叼着一个大烟斗，戴顶鸭舌帽，上面几个大字 《福尔摩斯侦探全集》，如获至宝，当下全搬出来，《回忆录》，《归来记》和 《最后致意》，这是一个系列，还有前面 3 本，不知道被收到哪里去了，软磨硬泡下，姐姐只好答应让我拿回家了，嘱托我好好珍惜，我回家后，看这几本书都要洗手先，每页慢慢的翻，不敢折角，不敢划线，岂料到，一天带到学校，被同学瞄到，只见他砸吧砸吧刚捏辣条的手，无情的拿起了书，书右侧边雪白的纸业上赫然显出一个油红色的手指印，从此我对该同学敬而远之。

后来姐姐外地工作，我又翻出来了另外 3 本，顺回囊中，心满意足。

初中后，到同学家里玩的次数多了，看到没见过的书手又痒痒了，但这毕竟不是主场了，于是改变策略，开始换书看，最深刻的是我拿了3册《中华上下五千年》简编版，换到了同学家的 《三国演义》和 《谁动了我的奶酪》，后面那本很久以后才看，而《三国演义》确是我读得最爽的一本书，一气呵成，这且听下回分解六个字永远有股魔力，拉着你往下看，躺在床上，整整三天，一字一句看完了整本，虽然看到关羽败走麦城以后就有些热血渐冷了。后来同学说那几本书被他爷爷拿走后找不到了，我也不想再还给他了，今年回家出门时也带上了这本 《三国》。他家还有整部的《千门》，可惜现在也不见了。

工作后，恰巧 微信读书 刚刚出来，我之前试用过很多读书软件，都没有它那样的简单，清爽，而又能和朋友一起分享读书想法，前路虽远，由可期许。每天地铁上的近一小时就大部分给了它。我个人还是更喜欢读纸质书的，读书，读固然重要，而精美的书肯定能加分的，装帧，纸张，排版，印刷，其实都能影响阅读的体验，当然氛围也很重要，我喜欢晚上读书，开上一盏小台灯，柔柔的灯光洒在字里行间，耳机里放着高山流水，莫大的享受。心血来潮就抄写抄写，也顾不得大煞风景的字迹了。

在深圳图书馆无意间看到了几本《冒险小虎队》, 最近读武侠又在杭州看到了《千门》，就像多年不见的老朋友一样，惊喜，满足。

其实很多时候读过的书，过了段时间后，就全然不记得了，或者朦朦胧胧，我却一直觉得，那时它给了我快乐，给了我慰藉，尽管我不大记得，但是他们已经是我整个人格的一部分了，他们让我更完整，更独立，更特别。

最近读爽过么？

PS: 图为 3 个正在刚刚被战火中毁坏的图书馆中读书
